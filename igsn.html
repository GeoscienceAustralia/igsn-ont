Abstract

This is an OWL ontology version of the International Geo Sample Number (IGSN) descriptive metadata XML schema (http://schema.igsn.org/description/) created by Geoscience Australia to describe samples in their Samples Register (http://pid.geoscience.gov.au/sample/). While it is based on the IGSN Schema, draws its structure and most of its formulation from several upper and domain OWL ontologies, namely PROV, Dublin Core, (upper ontologies) and SAM-LITE, GeoSPARQL (geo/spatial domain) ontologies. All but one od the classes declared in this ontology provide superclasses for all of the controlled code lists of metadata terms defined in the IGSN Code Lists vocabulary (http://pid.geoscience.gov.au/def/ont/ga/igsn-codelists). The single other class declared is 'Sample' which is an equivalent class of SAMFL's Specimen but is decared in order to impose a number of constraints on 'Samples' in order to ensure minimal metadata. Since all of the classes and properties used in this ontology draw from those previously mentioned upper ontologies and foundational OWL ontologies such as RDF, RDFS and OWL itself, it is possible to detail much information for Samples using them in common patterns. For example, declaring multiple related people to a Sample can be done by using PROV-O's prov:qualifiedAttribution pattern with different roles declared for those Agents (People/Orgs). This reuse of existing ontologies is desirable as it is likley that they are widely understood and well formed.
